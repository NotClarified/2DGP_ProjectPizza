2주차 구현 과제 - 피자 도우 토핑 및 자르기 구현
1-0. 도우를 누르면 도우가 판 위에 올라가도록 구현
1-1. 치즈가루를 누른 후 마우스를 누른 상태일경우 0.3초마다 해당 치즈가루가 피자 위에 쌓이도록 구현
1-2. 토마토 소스를... 위와 코드 동일
1-3. 토핑 더미를 위에 누른 후 마우스를 클릭할 경우 해당 위치에 토핑 더미를 올림.
1-4 토핑의 경우 랜덤한 각도로 누우며, 각 프레임마다 조금씩 커졌다 작아졌다 하면서 움직이도록 구현

주의해야하는 내용 - 해당 내용은 모두 피자 원형판 위에서 진행해야 하며, 피자 판 외의 다른곳을 클릭했을 경우 눌러도 처리되지 않도록 구현해야함


3주차 구현과제 - 피자 자르기 미니게임, 포장 미니게임 
1. 피자 전체를 클릭 후 오븐으로 드래그,드랍을 통해 굽기.
2. 5초 후에 도우, 소스 이미지 교체해서 나옴
3. 도마 위로 위치 이동
4. 점선을 놓고, 스페이스바를 누르면 메이플 스타캐치 미니게임 진행
